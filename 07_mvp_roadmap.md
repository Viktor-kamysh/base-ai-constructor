# Construction Pro Agent — MVP Roadmap

## Phase 1 — Ship first usable version
### Goal
Launch a useful assistant that saves time immediately.

### Scope
- Project workspace
- File uploads
- Voice note intake
- Site diary draft generation
- PDF/DOCX/email export
- Branding with logo/signature/stamp
- Estimate parsing
- Basic estimate review
- Version history
- Issue log

### Do not include yet
- full automatic quantity takeoff
- deep drawing-to-estimate mapping
- autonomous final estimate approval
- uncontrolled web pricing

## Phase 2 — Improve quality and value
### Scope
- licensed external pricing source integration
- rule engine activation
- stronger estimate comparison
- issue lifecycle improvements
- system-by-system review mode

## Phase 3 — Advanced pro workflows
### Scope
- deeper documentation-assisted completeness review
- tender comparison
- change order assistant
- weekly/monthly roll-up reporting

## QA before release
- test on 10+ real site diary cases
- test on 10+ real contractor estimates
- create gold-standard reviewed outputs
- track false positives and false negatives
- verify missing-data behavior

## Product promise at launch
"Construction Pro Agent helps professionals draft site diaries, review contractor estimates, keep project memory, and prepare structured outputs faster. All outputs remain draft-for-review."
